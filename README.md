# 🍽️ Store Service - Restaurant Reservation System

## 📌 Overview
Store Service is a full-stack web application designed to manage restaurant reservations and menu operations.  
The system allows customers to book tables online while enabling restaurant staff to efficiently manage reservations and menu items.

---

## 🚀 Features

### 👤 Customer Side
- Make table reservations online
- View available time slots
- Browse restaurant menu

### 🧑‍💼 Admin / Staff Side
- Manage reservations (create, update, delete)
- Manage menu items
- View incoming reservation requests

---

## 🛠 Tech Stack

### Frontend
- Angular

### Backend
- Spring Boot (REST API)

### Database
- PostgreSQL

### Tools
- Fork
- DBeaver (Database Management)
- Swagger (API Testing)

---

## 🏗️ System Architecture
This project follows a typical client-server architecture:

- Frontend communicates with backend via RESTful APIs
- Backend handles business logic and database operations
- Database stores reservation and menu data
